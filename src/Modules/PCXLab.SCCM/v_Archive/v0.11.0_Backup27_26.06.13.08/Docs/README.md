# PCXCMTools

Professional PowerShell automation toolkit for Microsoft Configuration Manager (SCCM / MECM).

## Features

- Create and organize Device Collections
- Move Configuration Manager objects into folders
- Standardized path resolution
- Reusable helper functions
- Modular PowerShell architecture
- Pester test support
